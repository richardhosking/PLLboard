Pseudo code 
For ADF4351 to derive settings depending on freq entered 
Aim for 1 KHz steps

Frequency in KHz entered on Pi touchscreen 
Calculate settings for output automatically

ADF4351 VCO 2.2 to 4.4GHz fundamental 
Reference clock on this board 25MHz
R Counter is reference clock divider range 1 to 1023
_divider is output divider - divides down VCO output 1 2 4 8 16 32 64 in binary steps 

    _feedbackType = 0;  // 0=fractional, 1=integer
    _ABP = 0;		// antibacklash pulse width 6ns for Fractional N mode 

    //    reference divider and multiplier both 1 
    _ref_doubler = 0; 	// reference doubler 1
    _ref_halve = 0; 	// reference divider 1
    
    R_counter = 25;  	// make INT step size 1MHz 
    _mod = 1000;	// Modulus of Frac N section 1000 to give step size of 1 KHz  
    
    if (_freq > 2200000) {
        _divider = 0; 		// data entered into register gives output_divider = 1;
     }
    else if (_freq > 1100000) {
        _divider = 1;           // divide output by 2;
          }
        else if (_freq > 550000) {
            _divider = 2;            // divide output by 4 
              }
            else if (_freq > 275000) {
                    _divider = 3;          // divide output by 8  
                  }
                else if (_freq >137500) {
                        _divider = 4;               // divide output by 16 
                     }
                    else if (_freq >68750) {
                            _divider = 5;               // divide output by 32 
                         }
                        else {  			//_freq lies between 34.375MHz to 68.75MHz
                            _divider = 6;               // divide output by 64 
                        }
 
// VCO Output division = 2^(_divider)
output_division = 2^(_divider)
Final freq = 1000*(INT + frac/mod)/(output division) KHz

//    quotient = dividend / divisor;
//    remainder = dividend % divisor;
// Hence 

INT = freq *output_division/1000 ; //quotient integer part 
frac = ((freq *output_division)%1000)*1000 ; // remainder X 1000
